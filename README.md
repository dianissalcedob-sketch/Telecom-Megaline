# Telecom-Megaline
# 📊 Análisis de Tarifas - Megaline

## 📌 Descripción del Proyecto

Este proyecto analiza el comportamiento de los clientes de la empresa de telecomunicaciones Megaline, que ofrece dos planes prepago: **Surf** y **Ultimate**.

El objetivo es determinar cuál de los planes genera mayores ingresos para apoyar la toma de decisiones del área comercial.

---

## 📂 Datos

El análisis se basa en información de 500 usuarios, incluyendo:

* Datos de usuarios
* Llamadas realizadas
* Mensajes enviados
* Uso de internet
* Información de planes

---

## ⚙️ Proceso de Análisis

1. Limpieza y preparación de datos
2. Agregación del consumo mensual por usuario
3. Cálculo de ingresos por usuario
4. Análisis exploratorio
5. Pruebas de hipótesis

---

## 📊 Resultados

* El plan **[Surf/Ultimate]** genera mayores ingresos promedio
* La diferencia es **[estadísticamente significativa / no significativa]**

---

## 🧪 Hipótesis

1. Diferencia de ingresos entre planes
2. Diferencia de ingresos entre usuarios de NY-NJ y otras regiones
